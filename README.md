# このプラグインについて
3月10日にこのプラグインの新バージョンを公開しました。
下記のURLからダウンロードして、前のファイルと置き換えてください。
しゆかつサーバーでBANされてる人があなたのサーバーに入れなくなるやつです<br>
ちなみに、ホワイトリストに入れとけばその人はあなたのサーバーに入れます<br>
